# Markdown 最佳实践

## 标题层级

```markdown
# 一级标题（文档标题）
## 二级标题（主要章节）
### 三级标题（子章节）
```

## 代码块

```python
# 指定语言
def hello():
    print("Hello, World!")
```

## 表格

| 列 1 | 列 2 | 列 3 |
|------|------|------|
| 数据 | 数据 | 数据 |

## 列表

### 无序列表
- 项目 1
- 项目 2
  - 子项目 2.1

### 有序列表
1. 第一步
2. 第二步

## 链接和引用

[链接文本](https://example.com)

> 引用内容

## 强调

**加粗** 或 __加粗__
*斜体* 或 _斜体_
~~删除线~~

## 最佳实践

1. **保持一致的标题层级**
2. **代码块指定语言**
3. **表格对齐列**
4. **避免过长的段落**
5. **使用语义化标题**
